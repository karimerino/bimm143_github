Carolina's Bioinformatics Portfolio

Welcome! This portfolio showcases my work from the Bioinformatics (Bimm 143) course at UC San Diego. Highlights include advanced analyses such as machine learning models, PCA for dimensionality reduction, RNA-Seq analysis, and protein structure prediction with AlphaFold.

Class 01: Introduction
Introduction to R and basic bioinformatics workflows.
Class 02: Data Manipulation and Plotting
Explored data manipulation and created basic plots in R to visualize biological datasets.
Class 03: Advanced Database Searching
Used NCBI databases and advanced search techniques to retrieve sequence data.
Class 04: Data Visualization
Created plots, boxplots, and exploratory visualizations using ggplot2 in R.
Class 05: Data Analysis
Performed statistical analyses including t-tests and ANOVA on biological datasets.
Class 06: Clustering & PCA
Applied clustering methods and PCA for dimensionality reduction to reveal patterns in complex datasets.
Class 07: Machine Learning 1
Built basic predictive models and explored classification techniques in R.
Class 08: Breast Cancer Analysis
Conducted differential expression analysis and visualized cancer gene datasets.
Class 09: Structural Bioinformatics Part 1
Analyzed protein structures and explored sequence alignments using computational tools.
Class 10: Halloween Mini Project
Applied previous skills in a creative mini-project integrating multiple datasets.
Class 11: Structural Bioinformatics Part 2
Extended structural analysis with visualization of protein complexes and AlphaFold predictions.
Class 12: RNA-Seq Analysis
Performed RNA-Seq preprocessing, normalization, and PCA to analyze transcriptomic data.
Class 14: RNASeq Mini Project
Conducted a complete RNA-Seq project including differential expression analysis, visualization, and pathway annotation.
